# Example Project

这个示例项目的目的不是展示最终设计，而是展示一套完整 Deck 工程在中间阶段应当包含哪些文件：

- `deck_brief.md`
- `deck_vibe_brief.md`
- `deck_hero_pages.md`
- `deck_clean_pages.md`
- `deck_visual_system.md`
- `deck_component_tokens.md`
- `deck_geometry_rules.md`
- `deck_page_skeletons.md`
- `deck_theme_tokens.json`
- `layout_manifest.json`
- `slide_state.json`

如果使用默认的 Expert Mode，还应关注：

- `interview_session.json`
- `interview_preparation.json`
- `deck_expert_context.md`
- `review_package.json` 中的 `expert_mode_summary`

新项目可以直接从这里理解输出物结构，而不是只看主文档。
